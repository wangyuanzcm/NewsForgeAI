# NewsForgeAI

NewsForgeAI 是一个基于人工智能的新闻视频自动生成系统，可以从飞书文档中提取新闻信息，自动下载相关视频素材，并使用 AI 进行智能剪辑，生成适合短视频平台的新闻视频内容。

## 特性

- 🤖 AI 驱动的内容分析和生成
- 📝 自动从飞书文档提取新闻信息
- 🎥 智能视频下载和处理
- ✂️ AI 辅助视频剪辑
- 🗣️ 自动语音合成
- 💬 智能字幕生成
- 🎨 多平台视频格式支持
